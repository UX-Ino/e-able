# e-able

웹 접근성 향상을 위한 전문 솔루션 랜딩 페이지 프로젝트입니다.

## 🚀 프로젝트 개요

이 프로젝트는 Vite와 React를 기반으로 구축된 현대적인 웹 애플리케이션입니다. 사용자에게 최적화된 경험과 시각적으로 뛰어난 디자인을 제공합니다.

## 🛠 기술 스택

- **프레임워크:** React 18
- **빌드 도구:** Vite 6
- **스타일링:** Tailwind CSS 4, Emotion
- **UI 컴포넌트:** Radix UI, Material UI (MUI)
- **애니메이션:** Framer Motion (Motion 12)
- **데이터 시각화:** Recharts
- **기타 주요 라이브러리:** Embla Carousel, Lucide React, React Hook Form

## 💻 시작하기

### 필수 요구사항

- Node.js (최신 LTS 버전 권장)
- npm 또는 pnpm

### 설치 및 초기화

```bash
# 의존성 설치
npm install
```

### 개발 서버 실행

```bash
# 로컬 개발 서버 시작
npm run dev
```

### 프로덕션 빌드

```bash
# 프로젝트 빌드
npm run build
```

## 📁 프로젝트 구조

- `src/app`: 핵심 애플리케이션 컴포넌트 및 로직
- `src/styles`: 전역 스타일 및 디자인 토큰
- `src/main.tsx`: 엔트리 포인트
- `public`: 정적 자원 (이미지, 폰트 등)
